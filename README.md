# LC09 E-commerce Website

A responsive static e-commerce website template built with HTML, CSS, and JavaScript.

## Overview

This project is a front-end e-commerce website mockup that includes:
- Home page with hero carousel and featured sections
- Product listing page
- Product details page
- Shopping cart page
- Login and signup pages

## Project Structure

- `index.html` — Home page
- `product.html` — Shop/product listing page
- `productDetails.html` — Product details page
- `cart.html` — Shopping cart page
- `login.html` — Login page
- `signup.html` — Signup page
- `css/styles.css` — Main stylesheet
- `js/index.js` — Basic interactive behavior (mobile menu and popup)
- `images/` — Image assets used by the template

## Features

- Responsive navigation menu
- Hero slider using Glide.js styles
- Interactive UI elements for mobile and desktop
- Reusable layout for shopping and product pages
- Simple popup functionality on page load

## How to Use

1. Open `index.html` in a web browser.
2. Navigate between pages using the site header links.
3. Use `product.html` and `productDetails.html` to explore product layout.

## Dependencies

This project uses external assets via CDN:
- Boxicons for icons
- Glide.js for the hero slider styles

## Notes

- This is a static front-end template and does not include a backend or real shopping cart logic.
- The `js/index.js` file handles the mobile menu toggle and popup display only.
